# MJSDS Dashboard Milestone Plan

## Current Milestone

### Name
Maintenance readiness and release clarity

### Objective
Keep the deployment and setup story clear enough that the dashboard remains easy to maintain.

## Workstreams

### 1. Setup Accuracy
- verify the setup guide matches current configuration needs
- remove stale instructions

### 2. Release Safety
- keep deployment behavior simple and repeatable
- clarify what changes require extra review

### 3. Product Boundary
- keep PM ownership outside the repo
- keep the repo focused on the dashboard itself

## Milestone Deliverables

- onboarding pack
- current maintenance guidance
- clearer release expectations

## Exit Criteria

- setup is understandable
- maintenance work has a clear rhythm
- future edits have lower deployment risk
